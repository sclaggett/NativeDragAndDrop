Native Drag and Drop and Large File Upload Library for the Flash Player

This repository contains the source code and compiled SWC for a library that adds native drag and drop and large file upload support to an Actionscript project. It accompanies a technical post:

http://www.sclaggett.com/Native-Drag-and-Drop-and-Large-File-Uploads-with-the-Flash-Player/
